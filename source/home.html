AI-Powered Wildlife Classification

We’re developing a wildlife monitoring system that is able to capture bio-photonic signals, detect them over significant distances, and automatically classify them through Machine Learning.

What are we doing?

A new system for monitoring wildlife

Our system is composed of a network of autonomous detectors sending data remotely to a database.

How are we doing this?

A novel way to detect wildlife

We’re developing a detector that uses properties of eye-shine to detect nocturnal wildlife. We’re also using Machine Learning technology to build a model and classify collected data.

Why are we doing this?

A solution for problems in ecology

Current monitoring techniques are labour-intensive, subjective and unreliable. Further, current approaches do not reflect data across a systematic level.
